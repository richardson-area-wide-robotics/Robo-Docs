# 3D Printing with a MakerBot

## Slicing 
:::warning

MakerBot Print is out of support as of March 1st 2022, and does not support Linux

:::

To slice models for printing on a Makerbot device, it is recommended to use [MakerBot Print](https://www.makerbot.com/makerbot-print/)


### Adding Models:
You can add models by Clicking on the `Project panel` and selecting `Add Models`

### Preparing to export
You can use a `Export only` printer to export models when using a USB Stick, or select a connected printer and export directly to it.



### Slicing
Press the red `Estimates and print preview ` button to get a idea on the path of the toolhead and how long the print will take.

### Exporting
Press the red `Export` button to open the Export menu. You can use a `Export only` printer to export models when using a USB Stick, or select a connected printer and export directly to it.
